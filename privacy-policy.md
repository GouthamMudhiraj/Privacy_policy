Privacy Policy – Email Writer Assistant

Email Writer Assistant does not collect, store, or sell any personal data.

This extension reads email content only when the user clicks the “AI Reply” button in Gmail. The content is securely transmitted to a backend service for the sole purpose of generating an AI-powered email reply.

Data Usage:

Email content is processed temporarily to generate responses.

No data is permanently stored.

No personal data is shared with third parties.

No data is sold.

Security:
All communication between the extension and backend server is encrypted using HTTPS.

User Control:
The extension only activates when the user interacts with it. It does not run background tracking or monitoring.
